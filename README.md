# Prisma with Node.js native Typescript support

Sample project to test Prisma with Node.js native Typescript support.

## Prerequisites

- Node.js 23.x or higher
- pnpm 10.x or higher
- Docker
- permission to run `./start-infra.sh`

## Setup

```bash
pnpm install
pnpm start:infra
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Run

```bash
pnpm start
```

## Docker

```bash
docker compose up --build
```
